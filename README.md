# AmneziaWG Proxy-Chain 🛡️

Продвинутое решение на базе протокола **AmneziaWG (WireGuard с защитой от DPI)** для создания устойчивой "proxy-цепочки" между серверами.

Проект позволяет развернуть два узла для прозрачного и селективного роутинга трафика:
1. **Входной узел (РФ)** принимает все зашифрованные подключения. Обрабатывает трафик к локальным ресурсам (например, Госуслуги, Кинопоиск) напрямую, без потери скорости, а запросы к заблокированным сайтам (и DNS) перенаправляет во второй туннель.
2. **Выходной узел (Зарубежный)** выпускает туннелированный трафик в глобальную сеть.
